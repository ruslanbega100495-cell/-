# CleanSPB — статический сайт клининга (СПб)

Это отдельный лендинг в папке `cleaning-site/` (не затрагивает ваш текущий `templates/index.html`).

## Как открыть

1) Откройте файл `cleaning-site/index.html` двойным кликом (локально в браузере).

2) Либо запустите локальный сервер (рекомендуется, чтобы `fetch` работал одинаково):

```powershell
cd "c:\Users\rusla\Desktop\проект\cleaning-site"
python -m http.server 8080
```

После этого откройте в браузере `http://localhost:8080`.

## Калькулятор и заявки (n8n)

В файле `cleaning-site/app.js` есть константа:

- `N8N_WEBHOOK_URL` — вставьте URL вашего webhook n8n, чтобы заявки уходили прямо в n8n.

Если webhook не указан, заявки сохраняются локально в `localStorage` (ключ `cleanspb_leads`) — чтобы вы могли проверить поток данных.
